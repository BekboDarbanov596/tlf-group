# Agro Care - AI Платформа для фермеров

Экосистема для фермеров, объединяющая планирование, диагностику и ветеринарную помощь.

## Как запустить

1. **Инициализация базы данных**:
   Так как проект использует SQLite, нужно создать таблицу из схемы. Выполните:
   ```bash
   sqlite3 database/database.sqlite ".read database/init.sql"
   ```
   Или запустите `php db_init.php` (если PHP доступен в консоли).

2. **Запуск сервера**:
   Используйте встроенный сервер PHP:
   ```bash
   php -S localhost:8000 -t public/
   ```

3. **Доступ**:
   Откройте `http://localhost:8000` в браузере.

## Особенности
- **Strict MVC**: Чистый PHP без фреймворков.
- **Premium UI**: Темная тема, Glassmorphism, GSAP анимации.
- **Security**: Argon2ID хеширование, CSRF защита, Audit Logging.
- **AI Modes**: 3 сценария ИИ-консультаций.
- **Vet Network**: Поиск ветеринаров по региону.
